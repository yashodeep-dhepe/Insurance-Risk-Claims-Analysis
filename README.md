# Insurance-Risk-Claims-Analysis
A Power BI dashboard analyzing insurance risk and claims data, uncovering trends across demographics, car attributes, education, marital status, and coverage zones. It highlights total policies, claim amounts, frequencies, and high-risk segments through interactive visuals, enabling data-driven decisions for underwriting and risk management.
